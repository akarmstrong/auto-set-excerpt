Auto set the excerpt on publish.

== Description ==

Auto set the excerpt on publish. Will contain the first 150 words of a post and exclude tags.

== Installation ==

1. Upload `auto-set-excerpt.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
